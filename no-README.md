### 1️⃣ Étape 1 : Obtenir une Session ID et configurer les variables
1. Obtenir la Session ID : [**Générer ma Session**](https://victorious-catriona-dsck-fb522085.koyeb.app/)
2. Générer vos variables (.env) : [**Générateur de .env**](https://victorious-catriona-dsck-fb522085.koyeb.app/env.html)

### 2️⃣ Étape 2 : Lien de Déploiement (À copier)
Si vous ne voulez pas utiliser les boutons, copiez ce lien dans votre dashboard Render/Koyeb :
- **Dockerfile** : 
```text
https://github.com/Dr-Djibi/Deploie-menma-md
```

### 3️⃣ Étape 3 : Variables d'environnement (.env)
Utilisez le [**Générateur de .env**](https://victorious-catriona-dsck-fb522085.koyeb.app/env.html) pour créer rapidement votre configuration, ou copiez et remplissez manuellement ces variables lors du déploiement :

```text
SESSION_ID=Menma_md_
NUMERO_OWNER=224XXXXXXXX
NOM_BOT=𝙼𝙴𝙽𝙼𝙰-𝙼𝙳
PREFIX=.
MODE=private
LECTURE_AUTO_STATUS=non
AUTO_LIKE_STATUS=oui
LIKE_STATUS_EMOJIS=💚,❤️,✨,🔥,🙌
ZONE_DE_TEMPS=Africa/Conakry
DATABASE_URL=
STICKER_PACKNAME=Menma-MD
STICKER_AUTHOR=Dr-Djibi
MENU=https://files.catbox.moe/h0va1p.jpg
```
